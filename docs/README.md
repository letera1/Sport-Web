## 🔧 Development Workflow

```bash
# 1. Clone and setup
git clone <repository-url>
cd live-football-scores
npm install
cp .env.example .env

# 2. Start development
npm run dev

# 3. Make changes and test
npm run lint
npm run build

# 4. Commit and push
git add .
git commit -m "feat: your feature"
git push origin feature-branch

# 5. Create pull request
```
